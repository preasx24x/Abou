What is DTECH?

DTECH is a student-first tech brand founded by Preasx24, operating several sub-platforms for youth empowerment: DTECH Services (marketplace), DTECH Info (Q&A and student help), DTECH HQ (utilities and bots), and DTECH Uni (APS estimator). It uses GitHub Pages, JSON search, local storage, and CI/CD techniques for efficient, fast, and no-backend deployments.

DTECH empowers South African students by giving access to tools that help them study better, earn on the side, understand university admission, and build their own platforms — all for free.

DTECH is not just one website — it’s a growing system.
Built by Preasx24, for students like you.
